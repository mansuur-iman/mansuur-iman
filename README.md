## Projects

### [Blog API](https://github.com/mansuur-iman/blog-api)
A REST API serving two independently deployed frontends — a reader client and an author dashboard — from one backend. Implements JWT auth, role-based access control, draft/publish workflow, and paginated endpoints.

> Node.js · Express · Prisma · PostgreSQL · JWT · bcrypt

### [Where's Waldo – Fullstack](https://github.com/mansuur-iman/wheres-waldo-fullstack) &nbsp;·&nbsp; [Live](https://wheres-waldo-fullstack.vercel.app/)
A fullstack game where the backend validates player guesses using coordinate comparison against stored character bounds — character positions are never sent to the client. Server enforces image progression and computes ranked leaderboards on field completion.

> Node.js · Express · Prisma · PostgreSQL · JWT · React · Monorepo
